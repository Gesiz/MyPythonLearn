RPC

```
客户端发送内容到消息队列A，服务端监听队列A里有内容后取内容并进行处理，处理好的内容不能发回到队列A。一旦发回到队列A，处理后的内容会重新被服务端提取并被反复处理，步入死循环
这个时候需要一个新的队列B，服务端将处理好的内容传到队列B，让客户端从队列B中取内容
在这里队列B就是远程过程调用（RPC）
```
